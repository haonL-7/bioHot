A Tiered Evidence Framework for Assessing Translational Readiness of Candidate Probiotics in Livestock: A Systematic Evidence Mapping with Methodological Development

Liu Haoan, Feng Wen*

Yulin University, Yulin, Shaanxi 719000, China · Correspondence: wfeng@yulinu.edu.cn

Abstract
Probiotic development in livestock is hindered by the absence of a standardized framework for assessing the strength of functional evidence. Researchers routinely conflate genus-level metagenomic associations with strain-level causal validation, leading to inflated translational expectations. We developed a six-tier evidence assessment framework (Levels 1a to 4) through an iterative consensus-building process involving three independent reviewers. The framework incorporates strain specificity and target-host validation as non-negotiable dimensions, with a novel transitional tier (Level 3.5) that deconvolves host-species transfer from strain-origin transfer. Applied through a systematic evidence mapping of Phascolarctobacterium succinatutens, we find that none of the three functional claims exceeds Level 3 (murine causal validation with human-derived strains). Level 3.5 and Level 4 remain entirely vacant across all claims. The framework converts "insufficient evidence" into a granular diagnosis of precisely which experimental data are missing for each functional claim.
Six-Tier Evidence Framework
TierNameDefinitionQuestion Answered
L1aMechanistic evidenceIn vitro pure-culture metabolic assays, ex vivo tissue/cell-based experiments, indirect mechanistic evidence from heterologous species within the same genus.Does this bacterium have the biochemical capacity?
L1bAssociation inferenceGenus-level co-occurrence network analyses, abundance-phenotype statistical associations, metagenomic functional inferences.Is there a statistical signal?
L2Moderate inferenceDirect validation of the effector molecule in a porcine model with mechanistic pathway data, but without strain-specific attribution.Does the effector mechanism operate in the target host?
L3Strong inferenceCausal validation through mono-colonization with this bacterium in murine models using human-derived strains.Can the bacterium execute the function in a mammalian gut?
L3.5Transitional validationMono-colonization using human-derived strains in a porcine model. Novel tier that deconvolves host-species transfer from strain-origin transfer.Does the mechanism transfer across host species?
L4Established evidenceMono-colonization in weaned piglets using a porcine-derived strain, with independent replication.Does the mechanism operate under ecologically realistic conditions?
Critical Epistemological Distinction: Levels 1–3 address mechanistic plausibility under controlled conditions, whereas Level 4 addresses translational efficacy under ecologically realistic conditions. These are qualitatively different scientific questions, not merely points on a continuum.
The Level 3.5 Innovation

When a probiotic candidate that works in murine models fails in the target livestock host, the failure cannot be attributed to either host-species differences or strain-origin differences because both variables changed simultaneously. Level 3.5 deconvolves these two variables by holding strain origin constant (human-derived) while varying host species (porcine). This tier has direct practical value for experimental design: before committing to costly porcine-derived strain isolation, a Level 3.5 trial using the readily available human-derived strain in a porcine model provides a critical go/no-go checkpoint.

Key Finding: Across all three functional claims for P. succinatutens (succinate scavenging, barrier maintenance, lipid reduction), Level 3.5 and Level 4 remain entirely vacant. Comparator mapping of F. prausnitzii and Roseburia spp. confirms this gap is systematic rather than candidate-specific.
Relationship to MS1

This framework is complementary to the Compartmentalized Co-Metabolism Network Model (MS1). MS1 provides a biological model of bidirectional host–microbiome interactions across three gut compartments with a four-dimensional matrix rating system for diagnostic evidence mapping. MS2 provides a hierarchical evidence assessment framework specifically for evaluating translational readiness of candidate probiotics. The two frameworks operate at different levels of analysis: MS1 at the metabolite-node level (what is the evidence distribution across a co-metabolism network?), and MS2 at the strain level (how close is a specific probiotic candidate to translational readiness?).
