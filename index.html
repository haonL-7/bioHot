Evaluation Methodology

The evidence assessment framework underpinning BioHot is derived from two methodological manuscripts by Liu Haoan and Feng Wen (Yulin University, 2025–2026):

MS1 Liu H, Feng W. "Compartmentalized Co-metabolism in the Porcine Gut: Systematic Evidence Mapping and a Theoretical Framework for Epigenetic–Microbiome Interactions." Defines the Compartmentalized Co-Metabolism Interaction Network Model and the Four-Dimensional Matrix Rating System (Forward Pathway, Reverse Pathway, Coupling Verification Depth, Measurement Depth). MS2 Liu H, Feng W. "A Tiered Evidence Framework for Assessing Translational Readiness of Candidate Probiotics in Livestock: A Systematic Evidence Mapping with Methodological Development." Defines the Six-Tier Evidence Framework (L1a–L4) with strain-specificity and target-host validation as core dimensions.

Literature evaluation is powered by the DeepSeek API, with GLM-4-Flash as fallback and local rule-based heuristics as final backup. All AI-generated assessments include confidence indicators and evaluation method attribution. Independent verification is recommended before citing.
